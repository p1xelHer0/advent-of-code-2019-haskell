### Running puzzles

Puzzles are numbered `{1..25}{ab}`

To run all puzzles:

```
stack run
```

#### Running a single puzzle:

```
./run solve 1a
```

Benchmark the puzzle:

```
./run bench 1a
```

Test the puzzle:

```
./run test 1a
```
